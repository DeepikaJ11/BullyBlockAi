# BullyBlock AI

## Project Overview
BullyBlock AI is a system that detects and analyzes cyberbullying and cyberhate content using Artificial Intelligence.  
It aligns with UN SDG 16 – Peace, Justice, and Strong Institutions.

## Steps to Execute the Project

### Prerequisites
- Node.js installed
- Python 3.x installed
- MongoDB or Firebase account
- Internet connection

### Setup Instructions
1. Clone or download this repository.
2. Open the `Source_Code` folder.
3. For backend:
   - Open terminal → `cd backend`
   - Run `npm install`
   - Start server → `node server.js`
4. For frontend:
   - Open terminal → `cd frontend`
   - Run `npm install`
   - Start app → `npm start`
5. Open your browser at `http://localhost:3000`

### Output
The dashboard will display the analyzed cyberbullying detection results.
